About

Your personal schedule tracker with dual timezone support

Purpose

A modern, glassmorphic schedule tracker designed to help students manage their daily school and tuition schedules with ease. Features dual timezone support for IST and WIB.

Features

Real-time schedule tracking, dual timezone display, category filtering, interactive UI with smooth animations, and a clean glassmorphic design that works perfectly on all devices.

Goal

To provide students with a simple yet powerful tool for managing their academic schedules, ensuring they never miss important classes or tuition sessions.

Key Features

Real-time Clock Display
Dual Timezone Support
Category Filtering
Glassmorphic Design
Responsive Layout
Smooth Animations

Built With

HTML5 CSS3 JavaScript Inter Font JetBrains Mono Glassmorphism CSS Grid Flexbox
